    15-213: Introduction to Computer Systems

    Fall 2016

    An introductory course for computer systems from the programmer's point of view. This course covers bit representations of data, machine-level code, memory hierarchy, cache, signals and processes, memory allocation, network programming, and concurrency.

    This is one of my favorite courses because it helps me understand the underlying systems of a computer from a person who knows nothing about how computer works. In this course I learned how to read assembly and made a cache simulator, a shell, an implementation of malloc, and a multi-threaded and cached proxy.

    Language Used: C

    15-251: Great Theoretical Ideas in Computer Science

    Spring 2016

    This course is about how to use theoretical ideas to formulate and solve problems in computer science. It integrates mathematical material with general problem solving techniques and computer science applications. Topics include algorithms, complexity theory, game theory, probability theory, graph theory, algebra, cryptography, and combinatorics.

    This is a really hard course, but I learned a lot about theorical computer science.

    15-150: Functional Programming

    Fall 2016

    An introduction to programming based on a functional model of computation. This course focuses on the datatype correctness, recursive functions, proof by structural induction, complexity of recurrences, structuring large programs into separate modules, and the integration of imperative programming.

    I really enjoyed this course and functional programming is really fun. Among my favorite topics are high-order function, continuation-passing style, minimax and alpha-beta pruning.

    Language Used: SML

    15-122: Imperative Programming

    Spring 2016

    The course teaches imperative programming and methods for ensuring the correctness of programs. Topics include theory and applications of important algorithms and data structures such as BFS, DFS, binary tree, AVL tree, heap, stack, queue.

    Language Used: C0, C

    15-112: Introduction to Fundamentals of Programming

    Fall 2015

    A technical introduction to the fundamentals of programming with an emphasis on producing clear, robust, and reasonably efficient code using top-down design, informal analysis, and effective testing and debugging. Starting from first principles, we will cover a large subset of the Python programming language, including its standard libraries and programming paradigms.

    My other favorite course at CMU. It is so good that I applied to be Teaching Assistant for it.

    Language Used: Python

    36-217: Probability Theory and Random Processes

    Fall 2016

    This course provides an introduction to probability theory. It is designed for students in electrical and computer engineering. Topics include elementary probability theory, conditional probability and independence, random variables, distribution functions, joint and conditional distributions, limit theorems, and an introduction to random processes. Some elementary ideas in spectral analysis and information theory will be given.

    21-127: Concepts of Mathematics

    Fall 2015

    This course introduces the basic concepts, ideas and tools involved in doing mathematics. As such, its main focus is on presenting informal logic, and the methods of mathematical proof. These subjects are closely related to the application of mathematics in many areas, particularly computer science. Topics discussed include a basic introduction to elementary number theory, induction, the algebra of sets, relations, equivalence relations, congruences, partitions, and functions, including injections, surjections, and bijections. A basic introduction to the real numbers, rational and irrational numbers.
